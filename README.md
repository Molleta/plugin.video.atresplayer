# AtrésPlayer Kodi Addon

## Overview
AtrésPlayer is an addon for Kodi that allows users to access and stream content from AtrésPlayer services seamlessly within the Kodi interface.

## Features
- Stream live TV channels
- Access to a vast library of on-demand content
- User-friendly interface for easy navigation
- Support for multiple languages

## Installation
To install the AtrésPlayer addon, follow these steps:
1. Open Kodi.
2. Go to `Add-ons` > `Install from repository`.
3. Select `AtrésPlayer Repository`.
4. Find and select the `AtrésPlayer` addon.
5. Click `Install` and wait for the addon to be installed.

## Usage
Once installed, you can access the AtrésPlayer addon from the `Add-ons` menu. Navigate through the categories to find your desired content.

## Troubleshooting
If you encounter any issues:
- Ensure that Kodi is updated to the latest version.
- Check your internet connection.
- Restart Kodi and try again.
